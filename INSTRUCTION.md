# Инструкция по добавлению текстового описания на страницы категорий

## Что было сделано:

### 1. Изменения в базе данных
Необходимо выполнить SQL-запрос для добавления нового поля `content` в таблицу `category`:

```sql
ALTER TABLE `category` ADD COLUMN `content` TEXT NULL AFTER `description`;
```

Файл с запросом: `/workspace/add_content_field.sql`

### 2. Изменения в коде

#### Модель (backend)
- **Файл:** `/workspace/app/models/admin/Category.php`
- **Изменения:** Добавлено сохранение поля `content` в методах `save_category()` и `update_category()`

#### Админ-панель (формы)
- **Файлы:** 
  - `/workspace/app/views/admin/Category/add.php`
  - `/workspace/app/views/admin/Category/edit.php`
- **Изменения:** 
  - Добавлено поле "Текстовое описание (контент)" с WYSIWYG-редактором CKEditor
  - Редактор позволяет форматировать текст, добавлять изображения, таблицы и т.д.

#### Публичная часть (отображение на сайте)
- **Файл:** `/workspace/app/views/Category/view.php`
- **Изменения:** Добавлен вывод текстового описания под заголовком H1 категории

## Как использовать:

### Для администратора сайта:

1. **Выполните SQL-запрос** через phpMyAdmin или другой инструмент управления БД:
   - Откройте файл `/workspace/add_content_field.sql`
   - Скопируйте SQL-запрос
   - Выполните его в базе данных сайта

2. **Добавьте описание категории:**
   - Зайдите в админ-панель: `/admin/category`
   - Выберите нужную категорию для редактирования (или создайте новую)
   - В форме редактирования появится новое поле **"Текстовое описание (контент)"**
   - Введите нужный текст, используя визуальный редактор (можно добавлять форматирование, изображения, ссылки и т.д.)
   - Нажмите "Сохранить"

3. **Результат:**
   - Текстовое описание отобразится на странице категории вида `/category/[slug]`
   - Описание будет показано сразу под заголовком H1 и перед списком брендов/товаров

### Пример:
Для страницы `/category/regulirovochnye-i-balansirovochnye-klapany`:
- Заголовок H1: "Регулировочные и балансировочные клапаны" (уже работает)
- Текстовое описание: контент, который вы добавите через админ-панель в поле "Текстовое описание (контент)"

## Технические детали:

- **Тип поля:** TEXT (позволяет хранить большой объем текста с HTML-разметкой)
- **Редактор:** CKEditor 5 (тот же, что используется для товаров и страниц)
- **Безопасность:** Контент выводится как есть (с HTML-тегами), так как заполняется через админ-панель доверенными пользователями
- **Отображение:** Если поле пустое - блок с описанием не показывается на странице

## Примечания:

- Поле `content` является необязательным (NULL)
- Можно добавлять описания только к тем категориям, где это необходимо
- Старые категории продолжат работать без изменений (просто без текстового описания)
