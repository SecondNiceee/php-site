# Инструкция по настройке блока "Вопросы-Ответы"

## 1. Выполните SQL-запрос в базе данных

Зайдите в phpMyAdmin и выполните следующий SQL-запрос:

```sql
CREATE TABLE IF NOT EXISTS `faq` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `entity_type` varchar(50) NOT NULL COMMENT 'Тип сущности: product, category, brand, main',
  `entity_id` int(11) NOT NULL COMMENT 'ID сущности (0 для главной)',
  `question` text NOT NULL,
  `answer` text NOT NULL,
  `sort_order` int(11) DEFAULT '0',
  `status` tinyint(1) DEFAULT '1',
  `created_at` datetime DEFAULT CURRENT_TIMESTAMP,
  PRIMARY KEY (`id`),
  KEY `entity` (`entity_type`, `entity_id`),
  KEY `sort` (`sort_order`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

## 2. Добавьте вопросы через админку

1. Зайдите в админку: `/admin`
2. В левом меню появится новый пункт **"Вопросы и ответы"**
3. Нажмите **"Добавить вопрос"**
4. Заполните поля:
   - **Вопрос** - текст вопроса
   - **Ответ** - развернутый ответ (можно использовать визуальный редактор)
   - **Тип сущности** - выберите где показывать вопрос:
     - Главная страница
     - Товар
     - Категория
     - Бренд
   - **Сущность** - выберите конкретную категорию/товар/бренд
   - **Порядок сортировки** - число, меньшее = выше в списке
   - **Активен** - галочка для отображения

## 3. Где будут отображаться вопросы

Блок "Вопросы-Ответы" автоматически появится на:
- Страницах категорий (после описания категории)
- Страницах товаров (внизу страницы)
- Страницах брендов (после списка товаров)
- Главной странице (после всех секций)

## 4. Особенности дизайна

- Блок выполнен в стиле сайта (цвета #082a43, #2f72cf)
- Аккордеон с плавной анимацией
- Полностью адаптивный дизайн для мобильных устройств
- При клике на вопрос он раскрывается, остальные закрываются
- Иконка "+" поворачивается при открытии

## 5. Управление вопросами

- Можно фильтровать вопросы по типу сущности и ID
- Можно редактировать и удалять вопросы
- Можно менять порядок сортировки
- Можно скрывать вопросы без удаления (снять галочку "Активен")
